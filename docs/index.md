## Quick Commands

```bash
rxiv init my-paper          # Create new manuscript
rxiv pdf                    # Generate PDF
rxiv validate              # Check manuscript
rxiv arxiv                 # Prepare arXiv submission
rxiv track-changes v1 v2   # Compare versions
```
